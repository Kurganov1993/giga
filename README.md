GigaChat Voice Assistant
Веб-приложение на Next.js, интегрированное с GigaChat API (Сбер) для общения с AI, а также с SaluteSpeech для синтеза речи. Поддерживает голосовой ввод через Web Speech API. Позволяет создавать несколько чатов, экспортировать/импортировать историю, оценивать ответы и настраивать параметры модели.

✨ Возможности
Чат с AI (GigaChat) с сохранением истории сессий

Голосовой ввод сообщений (Speech-to-Text) через встроенный Web Speech API

Озвучивание ответов (SaluteSpeech) — естественные голоса от Сбера

Настройка температуры и режима потоковой передачи (streaming)

Управление чатами (создание, удаление, переименование, поиск)

Экспорт/импорт сессий в JSON

Оценка ответов (лайк/дизлайк) для сбора обратной связи

Адаптивный дизайн (Tailwind CSS)

Индикация загрузки, записи, озвучивания

🛠 Технологии
Frontend: Next.js (App Router), React, TypeScript, Tailwind CSS, Lucide Icons

Backend: Next.js API Routes

AI API: GigaChat (Sber), SaluteSpeech (Sber)

Дополнительно: Web Speech API, UUID, Axios

📋 Предварительные требования
Node.js 18+ и npm/yarn

Аккаунт на developers.sber.ru с доступом к GigaChat и SaluteSpeech

Полученные client_id и client_secret для обоих сервисов
